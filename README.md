📊 Students Behaviour and Score Analysis Dashboard
📊 Project Overview

This project is an interactive Power BI dashboard that analyzes student performance using scores, attendance, and behaviour data. It helps identify academic trends, compare performance across subjects and terms, and support better decision-making in education.
